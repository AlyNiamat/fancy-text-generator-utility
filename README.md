# 🌟 Fancy Text Generator & Unicode Font Utility

Welcome to the **Fancy Text Generator Utility**! This open-source JavaScript library allows developers and creators to instantly convert standard ASCII text into beautiful, stylized Unicode characters (such as bold, italic, cursive, and gothic styles) for social media, gaming profiles, and aesthetic bios.

## Features
- 🚀 Lightweight and zero-dependency.
- ⚡ Real-time Unicode transformation.
- 🎨 Supports multiple aesthetic text styles.
